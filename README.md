# Tutorial: Teoria de Grafos e Grafos de Conhecimento com Oracle Autonomous Database

Este tutorial explora os fundamentos da teoria de grafos, os conceitos de grafos de conhecimento e sua implementação utilizando o Oracle Autonomous Database com a **Property Graph Query Language (PGQL)**. Além disso, apresentamos uma implementação em Python para extrair relacionamentos de documentos por meio de Modelos de Linguagem de Grande Escala (LLMs) e armazená-los como estruturas de grafos no Oracle.

## O que é um Grafo?

Um grafo é uma estrutura fundamental na matemática e na ciência da computação, usada para modelar relações entre objetos. Ele é composto por:

- **Vértices (Nós):** Representam entidades, como pessoas, lugares ou objetos.
- **Arestas (Links):** Representam os relacionamentos entre essas entidades.

Grafos são amplamente aplicados em diversas áreas, como redes sociais, redes semânticas e grafos de conhecimento. Para facilitar a visualização de forma intuitiva e didática, utilizamos o **3D Networking Graph** para representar nossos grafos neste tutorial.

## O que é um Grafo de Conhecimento?

Um grafo de conhecimento é uma representação estruturada de informações do mundo real, onde:

- **Nós:** Representam entidades, como pessoas, locais, produtos, etc.
- **Arestas:** Representam relações semânticas entre essas entidades, como "trabalha em", "faz parte de", entre outros.

Esses grafos são poderosos para melhorar a pesquisa semântica, sistemas de recomendação e aplicações de resposta a perguntas. Neste tutorial, mostramos como combinar **RAG (Retrieval-Augmented Generation)** com **Vector Search** e **Property Graph** para criar recomendações personalizadas de filmes.

## Objetivos do Tutorial

- Entender os conceitos básicos de grafos e grafos de conhecimento.
- Aprender a utilizar o Oracle Autonomous Database e PGQL para manipular estruturas de grafos.
- Explorar uma implementação prática em Python para extrair e armazenar relacionamentos de dados.
- Visualizar grafos de forma interativa com ferramentas 3D.

Fique à vontade para explorar o código e os exemplos práticos fornecidos!
